# getSwap

Action for fetching the optimal swap route between two tokens and then providing additional processed fields relevant to the swap being performed.

## Initialization

Ensure you have followed the steps from [Actions > Getting Started](/actions/getting-started) before getting started

## Import

```ts
import { getSwap } from "@bera/berajs/actions";
```

## Required config

For more information on BeraConfig, see [BeraConfig](/bera-config/)

```ts twoslash
import { type BeraConfig } from "@bera/berajs";

// required configs for getSwap
export const beraConfig: BeraConfig = {
  endpoints: {
    dexRouter: "https://...",
  }, 
};
```

## Usage

```ts
import { type BeraConfig } from "@bera/berajs";
import { getSwap } from "@bera/berajs/actions";

// required configs for getSwap
export const beraConfig: BeraConfig = {
  endpoints: {
    dexRouter: "https://...",
  }, 
};

const swap = await getSwap({
  args: {
    tokenIn,
    tokenOut,
    tokenInDecimals,
    tokenOutDecimals,
    amount,
  },
  config: beraConfig,
  publicClient,
});
```

## Parameters

| Name                    | type                              | Description                                                   | Required |
| ----------------------- | :-------------------------------- | :------------------------------------------------------------ | :------: |
| `args.tokenIn`          | `string`                          | The address of the token to be given in the swap              |  `true`  |
| `args.tokenOut`         | `string`                          | The address of the token to be received in the swap           |  `true`  |
| `args.tokenInDecimals`  | `number`                          | The amount of decimals for the input token                    |  `true`  |
| `args.tokenOutDecimals` | `number`                          | The amount of decimals for the output token                   |  `true`  |
| `args.amount`           | `string`                          | The amount of the tokenIn to be given in the swap             |  `true`  |
| `config`                | [`BeraConfig`](/types/BeraConfig) | BeraConfig object containing addresses for relevant resources |  `true`  |
| publicClient            | PublicClient                      | The public client used to interact with the blockchain        |  `true`  |
